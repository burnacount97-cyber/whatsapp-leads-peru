== Installation ==

= Automatic Installation (Recommended) =

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for "**LeadWidget**"
3. Click **Install Now**
4. Click **Activate**
5. Go to **LeadWidget** in the sidebar
6. Enter your User ID (get it from [your dashboard](https://leadwidget.com/dashboard))
7. Click **Save Settings**

Your widget is now live!

= Manual Installation =

1. Download the plugin ZIP file
2. Go to **Plugins > Add New > Upload Plugin**
3. Choose the ZIP file and click **Install Now**
4. Activate the plugin
5. Follow steps 5-7 from automatic installation above

= Getting Your User ID =

1. Create a free account at [LeadWidget.com](https://leadwidget.com/register)
2. Go to your [Dashboard](https://leadwidget.com/dashboard)
3. Navigate to **Widget** tab
4. Copy your **User ID** from the installation code section
5. Paste it in the plugin settings
